## ‚öôÔ∏è Quick Start & Usage

This template is open source! You are free to clone, modify, and use this code for your own personal portfolio.

### 1. Clone the Repository
Run the following command in your terminal to download the code:

```bash
git clone https://github.com/mabdullahchaudhary/maida-portfolio.git
```

### 2. Open the Project
Since this project uses standard HTML, CSS, and JS, no package installation is required. 
* Simply open the `index.html` file directly in your web browser.
* **Pro-tip:** For the best development experience, use the **Live Server** extension in VS Code.
